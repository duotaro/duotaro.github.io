localStorageで保持し、必要であればfirebase storeに同期できるようにする。
```

■ 最重要KPI（読者価値）
□ 平均滞在時間: __分__秒 (目標: 3分以上)
□ 直帰率: __%  (目標: 60%以下)
□ リピート率: __%  (目標: 20%以上)
□ 内部リンククリック率: __%  (目標: 15%以上)
□ コメント・問い合わせ数: __件  (目標: 5件以上)
□ SNSシェア数: __件  (目標: 10件以上)

■ 二次KPI（サイト成長）
□ 月間PV数: __PV  (前月比: __%)
□ 検索流入割合: __%  (目標: 60%以上)
□ 検索順位向上記事数: __記事  (目標: 5記事以上)
□ 改修完了記事数: __本  (目標: 12本)
□ 新規記事公開数: __本  (目標: 8本)
```
